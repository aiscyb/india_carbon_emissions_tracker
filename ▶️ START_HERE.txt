═══════════════════════════════════════════════════════════════════════════════
  🐛 IF SOMETHING GOES WRONG
═══════════════════════════════════════════════════════════════════════════════

Problem: "Python is not recognized"
Fix: Install Python from https://www.python.org/downloads/

Problem: "pip is not recognized"  
Fix: Reinstall Python and check "Add Python to PATH" during installation

Problem: "Module not found"
Fix: Run this in Command Prompt:
  pip install pandas numpy requests scikit-learn anthropic streamlit plotly streamlit-folium folium

Problem: "Port 8501 already in use"
Fix: Run with different port:
  streamlit run app.py --server.port 8502

Problem: Data files not created
Fix: Run manually:
  python data_fetcher.py
  python ml_model.py
