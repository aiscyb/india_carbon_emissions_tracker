1. DATA FETCHER MODULE (data_fetcher.py)

                         TROUBLESHOOTING GUIDE
═══════════════════════════════════════════════════════════════════════════════

ISSUE 1: Python Not Found
──────────────────────────
Error: "python is not recognized as an internal or external command"
Cause: Python not installed or not in PATH
Solution:
  1. Download from https://www.python.org/
  2. Run installer
  3. CHECK "Add Python to PATH" during installation
  4. Restart Command Prompt
  5. Verify with: python --version

ISSUE 2: Module Not Found
──────────────────────────
Error: "ModuleNotFoundError: No module named 'pandas'"
Cause: Dependencies not installed
Solution:
  1. Open Command Prompt
  2. cd c:\Users\KIIT0001\OneDrive\Desktop\tryingit
  3. pip install --upgrade pip
  4. pip install -r requirements.txt
  5. Try again

ISSUE 3: Download Fails
──────────────────────────
Error: "Connection error" or "Timeout" when downloading OWID data
Cause: Network issue or firewall blocking
Solution:
  1. Check internet connection
  2. Disable VPN if active
  3. Check firewall settings (may need to allow Python)
  4. Try again after waiting 5 minutes
  5. Use mobile hotspot as backup

ISSUE 4: Port Already in Use
──────────────────────────────
Error: "Address already in use" when starting Streamlit
Cause: Another application using port 8501
Solution:
  1. Kill previous Streamlit process
  2. Or use different port: streamlit run app.py --server.port 8502
  3. Access at: http://localhost:8502

ISSUE 5: Permission Denied
──────────────────────────────
Error: "Permission denied" creating files
Cause: Insufficient permissions on directory
Solution:
  1. Run Command Prompt as Administrator
  2. Or move project to different location with write access
  3. Try again

ISSUE 6: Insufficient Disk Space
──────────────────────────────────
Error: "Disk space exhausted" or similar
Cause: Not enough free space
Solution:
  1. Free up at least 1 GB on disk
  2. Delete temporary files if needed
  3. Try again
