═══════════════════════════════════════════════════════════════════════════════
                        🆘 COMMON ISSUES & FIXES
═══════════════════════════════════════════════════════════════════════════════

ISSUE: "Python is not recognized"
FIX:   • Install Python from https://www.python.org/
       • Check "Add Python to PATH" during installation
       • Restart Command Prompt

ISSUE: "ModuleNotFoundError"
FIX:   • Run: pip install -r requirements.txt
       • Wait for installation to complete
       • Try again

ISSUE: "Download fails" or "Connection timeout"
FIX:   • Check internet connection
       • Disable VPN if active
       • Check firewall settings
       • Try again after 5 minutes

ISSUE: "Permission denied" or "Access denied"
FIX:   • Run Command Prompt as Administrator
       • Or move project to C:\Users\YourName\Documents\
       • Try again

ISSUE: "Port 8501 already in use"
FIX:   • Use different port: streamlit run app.py --server.port 8502
       • Then access: http://localhost:8502

For more help, see EXECUTION_GUIDE.md or EXECUTION_SUMMARY.txt
